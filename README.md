# 💬 Chat Application

A responsive chat interface built to simulate real-time messaging using web technologies.

## 🚀 Features
- User authentication (login/signup)  
- Messaging interface  
- Clean and responsive UI  
- Real-time-like message updates using AJAX (without page refresh)      

## 🛠️ Technologies Used
- HTML  
- CSS  
- PHP  
- MySQL  

## 💡 Description
This project focuses on building a chat system with a clean user interface and smooth user experience.  
Implemented asynchronous message updates using AJAX to simulate real-time interaction without requiring page reloads.
